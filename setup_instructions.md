# Setup Instructions

## Requirements
- Python 3.8+
- GPU recommended (trained on Kaggle)

## Installation
pip install -r requirements.txt

## Dataset Paths (Kaggle)
- sarscov2-ctscan-dataset
- mosmeddata-chest-ct-scans-with-covid19
- ct-tuberculosis-segmentation
- covid19-radiography-database
- combined-unknown-pneumonia-and-tuberculosis

## How to Run
1. Upload the notebook to Kaggle
2. Attach the above datasets
3. Run all cells in order
